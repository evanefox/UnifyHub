# Plugin Manager UI

⚠️ **WARNING**: This is a core plugin. Do not remove, disable, or modify its files. Doing so can cause UnifyHub to become unstable, crash, or corrupt your Unity Hub installation.

## Overview

This plugin adds the Plugins management page to Unity Hub.

It lets UnifyHub users:

- View installed plugins
- Install plugin folders
- Enable or disable plugins
- Apply plugin changes
- Restore Unity Hub from backup
- Remove non-core plugins
